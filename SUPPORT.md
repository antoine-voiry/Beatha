# Support

## Need Help?
If you are having trouble with Project Beatha, here are your options:

1.  **Read the Docs:** Check `README.md` and `WIRING.md` first.
2.  **Search Issues:** See if someone else has reported the same bug.
3.  **Open an Issue:** If you found a bug, file a report using the Bug Report template.
